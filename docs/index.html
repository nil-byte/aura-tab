A customizable New Tab page for Chrome/Edge with smart backgrounds, quick links management, and immersive photo viewing. 为 Chrome/Edge 浏览器打造的精美新标签页，支持智能壁纸、快捷链接管理和沉浸式照片浏览。

Install from Chrome Web Store 从 Chrome 应用商店安装 View on GitHub 在 GitHub 上查看

Packed with Features 功能丰富

Everything you need for a beautiful and productive new tab experience. 打造美观高效新标签页体验所需的一切。

Smart Backgrounds智能背景系统

Auto-crop based on screen size with focal point detection and smart aspect ratio adaptation.根据屏幕尺寸自动裁剪，焦点检测，智能宽高比适配。

Multiple Sources多源支持

Support for local files, Unsplash, Pixabay, and Pexels with configurable rotation.本地文件、Unsplash、Pixabay、Pexels 多平台支持，可配置轮播。

Quick Links Manager快速链接管理器

Folder support, drag-and-drop sorting, instant search, pagination, and pin to dock.文件夹支持、拖拽排序、即时搜索、分页、固定到 Dock。

Bookmark Import书签导入

One-click import from Chrome bookmarks with automatic deduplication.一键从 Chrome 书签导入，自动去重。

Launchpad Mode启动台模式

macOS-style application launcher with folder organization.macOS 风格的应用启动器，支持文件夹组织。

Photo Viewer沉浸式照片查看器

Fullscreen browsing with keyboard navigation and EXIF display.全屏浏览、键盘导航、EXIF 信息显示。

Smooth Transitions平滑过渡动画

Beautiful fade animations when switching backgrounds with configurable interval.切换背景时淡入淡出，可配置切换间隔。

i18n Support国际化

Full Chinese (Simplified/Traditional) and English localization.完整的中英文（简/繁）支持。

Performance First性能优先

First Paint optimization, background caching with TTL, lazy loading.首屏优化、背景缓存、TTL 管理、懒加载。

Privacy Focused隐私保护

WebDAV backup for Nutstore, Synology NAS and more. No data collection.支持 WebDAV 备份（坚果云、群晖等），不收集任何数据。

Toolbar Customization工具栏图标定制

Upload and apply your own toolbar icons with instant preview.上传并应用自定义图标，实时预览。

Flexible Settings灵活设置

Background blur effects, clock styles, search engine customization and more.背景模糊效果、时钟样式、搜索引擎自定义等。
